###Swipable是什么  
Swipable实现了类似于网易新闻的标签选择页，只需要几行代码便可以集成,具体的使用方法参考Demo。  

---  
###CocoaPods  
你可以在 Podfile 中加入下面一行代码来使用YTKNetwork
```
pod 'TabSwipable', '0.0.2'  
```
